# 1. PROJECT OVERVIEW

## Objective
To design and develop a **Real-time Web Dashboard** for monitoring a smart warehouse system utilizing **4-way Shuttle** and **Lifter** technology.

## Scope
The system serves as the central Operational Dashboard, enabling warehouse managers to:
1.  Visually track the real-time position and status of all equipment within the grid.
2.  Monitor the progress of inbound/outbound tasks (Orders).
3.  Receive instant alerts regarding faults, errors, or connectivity issues.

## Core Requirements
* **Low Latency:** System response time < 500ms.
* **Visualization:** High-fidelity 3D rendering of the warehouse layout.
* **Infrastructure:** Direct communication with existing Siemens S7 PLCs (No intermediate Gateway).
